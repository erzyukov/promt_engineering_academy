# Prompt Engineering Academy — План проекта

Образовательный портал по промпт-инжинирингу, где **теория переплетена с практикой**: читаешь абзац — тут же проверяешь понимание интерактивным упражнением, идёшь дальше. В конце статьи — общий тест по всему материалу.

Этот каталог `docs/` — **пошаговый план реализации**. Сам код портала появится в `src/` по мере выполнения дорожной карты.

---

## Зафиксированные решения

| Параметр | Решение |
|---|---|
| **Стек** | React 18 + TypeScript + Vite |
| **Контент-формат** | MDX (Markdown + встроенные React-компоненты) |
| **Роутинг** | `HashRouter` (работает на GitHub Pages без серверных rewrite'ов) |
| **Деплой** | GitHub Actions → GitHub Pages (`https://erzyukov.github.io/promt_engineering_academy/`) |
| **Язык** | Русский — интерфейс и статьи (идентификаторы в коде — английские) |
| **Упражнения** | Полностью статические, **без LLM**. Проверка на клиенте: точные ответы + эвристики |
| **Обратная связь** | После ответа сразу показываем «верно / неверно» + короткое объяснение |
| **Контент** | 3 статьи; в каждой несколько тематических инлайн-тестов + финальный тест по статье |
| **Формат плана** | Поэтапно: MVP → полный. Статья №1 расписана целиком, №2 и №3 — тезисно |

## Что строим (в двух словах)

1. **Движок упражнений** — единый «контракт» компонента: получает данные, управляет состоянием (не отвечено / верно / неверно), сообщает результат наружу.
2. **9 типов упражнений** — 5 базовых (`MultipleChoice`, `TrueFalse`, `FillTheBlank`, `MatchPairs`, `OrderSteps`), 3 продвинутых (`SpotTheHallucination`, `PromptBuilder`, `CaseStudy`) и 1 кастомный (`PromptRefinementLadder`).
3. **Два режима** — *инлайн* (автономный виджет в тексте, без общего счёта) и *тест* (последовательность с прогресс-баром и итоговым результатом). Один и тот же компонент работает в обоих.
4. **Контент** — 3 статьи в MDX со встроенными упражнениями.
5. **Деплой** — автопубликация на GitHub Pages при пуше в `main`.

## Definition of Done проекта

- [x] Портал открывается по адресу GitHub Pages, навигация между статьями работает.
- [x] Все 9 типов упражнений работают в инлайн-режиме и показывают «верно/неверно» + объяснение.
- [x] Тест-режим: прогресс-бар, подсчёт баллов (с частичной правильностью где уместно), итоговая сводка.
- [x] 3 статьи опубликованы: в каждой ≥ 3 инлайн-теста разных типов + финальный тест.
- [x] Кастомный компонент `PromptRefinementLadder` реализован и встроен в статью.
- [x] Прогресс пользователя сохраняется в `localStorage`.
- [x] Адаптивная вёрстка (десктоп + мобильный), базовая доступность с клавиатуры.
